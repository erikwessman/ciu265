## Pre-requisites

- NodeJS 18+

## Environment Variables

Create a `.env` file in the root directory with the following variables:

```env
CONNECTION_STRING: <connection string to mongodb>
```

## Start

### Windows

Run `start.bat`.

### Manual

#### Backend

```bash
npm i && npm run start
```

#### Frontend

```bash
npm i && npm run dev
```
